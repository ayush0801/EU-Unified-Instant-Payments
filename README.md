# EU Unified Instant Payments App

## Overview
This project designs a BHIM-like unified instant payments application for the European Union, enabling alias-based P2P and merchant payments on top of SEPA Instant and Open Banking.

## Problem
Despite SEPA Instant availability, EU payments remain fragmented across banks with IBAN-heavy UX and inconsistent experiences. Users still rely on third-party apps for simple money transfers.

## Solution
A bank-agnostic payments app that:
- Links multiple bank accounts
- Enables alias-based payments (phone/email)
- Supports QR-based merchant payments
- Provides unified transaction history
- Improves failure transparency

## Key Features
- P2P instant payments
- P2M QR payments
- Alias mapping system
- Open Banking integration
- Unified UX layer
